# FredericoLabs

Sistema de Controle de Acesso Inteligente

Projeto baseado em:

* ESP32
* Bluetooth
* Biometria Android
* MIT App Inventor
* Display TFT
* Identidade Digital Ativa

## Objetivo

Criar um sistema moderno de autenticação para laboratórios e ambientes controlados, substituindo cartões físicos por autenticação biométrica integrada ao smartphone.

## Estrutura

### APP

Aplicativo Android responsável por:

* autenticação biométrica;
* comunicação Bluetooth;
* interface do usuário.

### ESP32

Controlador principal responsável por:

* receber autenticação;
* liberar acesso;
* controlar indicadores visuais;
* exibir informações no display TFT.

### Hardware

Documentação elétrica e mecânica.

### Assets

Imagens, mascote, logotipos e elementos visuais oficiais.
